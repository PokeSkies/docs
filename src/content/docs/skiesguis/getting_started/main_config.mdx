The guide explains configuring the main `config.json` file for SkiesGUIs. The main configuration file contains global settings that affect the entire mod.

<FileTree>
- config
    - skiesguis

</FileTree>

***
## Example File
> ```json
> {
>   "debug": false
> }
> ```

## Options
### Debug
Toggles the debugging status to make finding mistakes easier. Actual errors with GUIs will still be displayed even if false. This is not recommended to leave enabled as it will display a lot of information. Can also be toggled using the command `/guis debug`.
>```json
> "debug": false
>```
