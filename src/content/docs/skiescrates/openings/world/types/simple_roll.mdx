---
title: Simple Roll Animation
description: Configuration options for the SIMPLE_ROLL world animation type
---

A simple animation where a single item is spun above the crate block before giving the reward.
```json
{
  "type": "SIMPLE_ROLL",
  "spin_count": 30,
  "spin_interval": 1,
  "start_delay": 0,
  "change_interval": 3,
  "change_amount": 1,
  "end_delay": 40,
  "sound": {
    "sound": "minecraft:block.note_block.xylophone",
    "pitch": 0.5,
    "volume": 0.5
  },
  "offset": {
    "x": 0.0,
    "y": 1.0,
    "z": 0.0
  },
  "hide_hologram": true
}
```

## Inherited
All World Animation Types inherit a base set of options. See [World Animation Options](/skiescrates/openings/world/world_opening/#world-animation-options) for more information.

## Spin Count
The total number of spins this spinner will perform before stopping.
```json
"spin_count": 51
```
## Spin Interval
The **starting** interval, in ticks, between each spin of this spinner.
```json
"spin_interval": 1
```
## Start Delay
The delay, in ticks, before this spinner starts spinning from the start of the animation.
```json
"start_delay": 0
```
## Change Interval
The number of ticks before increase the [Spin Interval](#spin-interval) by the [Change Amount](#change-amount), in a loop.
```json
"change_interval": 5
```
## Change Amount
The amount, in ticks, to increase the [Spin Interval](#spin-interval) by every [Change Interval](#change-interval).
```json
"change_amount": 1
```
## Sound
The sound to play every time this spinner spins.
```json
"sound": {
  "sound": "minecraft:block.note_block.xylophone",
  "pitch": 0.5,
  "volume": 0.5
}
```
## Offset
The offset, using an XYZ floating point object, from the crate block to spawn the item at, relative to the bottom center of the block.
```json
"offset": {
  "x": 0.0,
  "y": 1.0,
  "z": 0.0
}
```
