General settings for the Preview.
```json
"settings": {
    "title": "Previewing %crate_name%",
    "menu_type": "GENERIC_9X6"
}
```
## Title
The title of the inventory GUI. Uses <a href="https://docs.advntr.dev/minimessage/format.html" target="_blank" rel="noopener noreferrer">MiniMessage formatting</a>!
```json
"title": "Previewing %crate_name%"
```
## Menu Type
The type of inventory menu to use.
```json
"menu_type": "GENERIC_9X6"
```
