## Commands
| Command                                   | Description                                                                              | Permission                        |
|-------------------------------------------|------------------------------------------------------------------------------------------|-----------------------------------|
| /actionitems                              | Default command                                                                          | actionitems.command.base          |
| /actionitems reload                       | Reload the Mod                                                                           | actionitems.command.reload        |
| /actionitems debug                        | Toggle the debug mode for more insight into errors                                       | actionitems.command.debug         |
| /actionitems give <id> [players] [amount] | Give players (or self) an action item                                                    | actionitems.command.give          |
| /actionitems resetcooldown <player> [id]  | Reset the cooldown for the specified player on an action item (or all if none specified) | actionitems.command.resetcooldown |
| /actionitems resetuses <player> [id]      | Reset the uses for the specified player on an action item (or all if none specified)     | actionitems.command.resetuses     |
