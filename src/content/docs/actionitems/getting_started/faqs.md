Due to the potential complexity of this mod, there are some frequent issues and questions that may arise when using ActionItems. Below are some of the most common questions and their answers:

### Player Heads
**Q:** How do I set a player head as an item in my GUI?

**A:** Player Heads can be set using the special keyword `playerhead` followed by an option specifier. See the [Player Heads](/actionitems/items/display_item/#player-heads) section in Supported Materials for more information on available playerhead item specifiers.
